# Investigate Workflow

## Objetivo
Root-cause debugging sistemático para encontrar causa raiz de problemas.

## Passos

1. **Definir o Problema**
   - Sintoma observado
   - Quando acontece
   - Frequência

2. **Hipóteses Iniciais**
   - Listar 3-5 causas possíveis
   - Priorizar por probabilidade

3. **Investigação**
   - Ler código relevante
   - Verificar logs
   - Isolar variáveis
   - Testar hipóteses

4. **Confirmação**
   - Reproduzir com causa raiz identificada
   - Verificar que fix resolve

5. **Documentação**
   - Causa raiz
   - Fix aplicado
   - Prevenção futura

## Output

```
Investigation: [Problem]

**Root Cause:**
- [Causa identificada em file:line]

**Fix:**
- [Solução aplicada]

**Prevention:**
- [Como evitar no futuro]
```
